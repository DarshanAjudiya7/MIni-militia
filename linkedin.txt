🚀 Exciting News! I’ve just finished building "Multi-Militia: Python Edition" — a tribute to the classic Mini Militia that we all loved! 🎮

This project has been an incredible journey into game development and real-time networking. It’s a 2D side-scrolling shooter that combines nostalgia with modern tech.

Key Technical Highlights:
✅ Core Engine: Built from scratch using Python & Pygame.
✅ Real-time Networking: Implemented a robust multiplayer system using Node.js and Socket.io for low-latency gameplay.
✅ Physics: Custom jetpack flight mechanics, gravity-based projectiles, and complex collision detection.
✅ Features: 15+ unique weapons, dynamic parallax backgrounds, and support for up to 4 local players or global online rooms.

Whether it’s sniping from across the map or dual-wielding SMGs while flying through the air, the gameplay is fast, fluid, and a lot of fun.

Building this challenged me to think deeply about state synchronization, efficient rendering, and game balance. Check out the source code and the new documentation I’ve put together!
